Registration

Eine Anmeldung für einen Wettkampf über das Internet ist nur dann möglich, wenn sich auch der Server im Internet befindet.

Für die Anmeldung sind die folgenden Daten erforderlich:

  • E-Mailadresse der Person, die die Anmeldung betätigt
    Diese ist für die eindeutige Identifikation und Mißbrauchvorsorge notwendig. Außerdem wird auf diese Adresse eine Anmeldungsbestätigung zugeschickt. Manche Servern verweigern leider diese Bestätigung anzunehmen.
  • Kampfrichter
    Falls die Wettkampfausschreibung es verlangt, werden hier die Kamprichter angemeldet.
  • Wettkampfbezeichnung
    Die Wettkampfbezeichnung wird aus der vorgeschlagenen Liste ausgewählt.
  • Wettkampfklasse
    Die Wettkampfklasse wird aus der vorgeschlagenen Liste ausgewählt.
  • Mannschaftsbeuzeichnung
    Falls es sich um einen Mannschaftswettkampf handelt, wird hier eine eindeutige Mannschaftsbezeichnung angegeben. Diese kann auch unterschiedlich von dem Sportverein sein.
    Bei einer Nachmeldung ist es notwendig, die gleiche Bezeichnung anzugeben.
    Bei den Wettkämpfen mit mehreren Runden (Liga), kann der Wettkampf bei einer Nachmeldung nur aus der vorgeschlagenen Liste gewählt werden. Eine Auflistung der bereits angemeldeten Teilnehmer kann mit einem Mausklick auf den Button "L" hervorgerufen werden.
  • Teilnehmerdaten
    Um die Telnehmer eindeutig identifizieren zu können, werden die Vor- und Nachname, der Geburtstag (JJJJ-MM-TT) und die Vereinszugehörigkeit verlangt. Dabei handelt es sich um die Vereinszugehörigkeit des Teilnehmers. Diese kann unterschiedlich von der Vereinszugehörigkeit der Mannschaft sein.
    Falls sich die Teilnehmerdaten bereits in Programm-Stammdaten befinden, werden diese beim Eingeben vorgeschlagen. Diese sind vorzugsweise zu wählen.

Es ist möglich, die Teilnehmer für verschiedene Wettkampfklassen oder mehrere Manschaften in einer Anmeldung zusamen zu fassen. Dafür muss einen neuen Tab eingefügt werden ("+" Button unter dem Absenden-Button). Vor dem endgültigen Absenden der Anmeldung ist es möglich, eine freie Textnachricht an die Wettkampfleitung beizufügen.
